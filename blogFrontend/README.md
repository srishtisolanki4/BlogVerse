# 💻 blogVerse — Frontend

The frontend of blogVerse built using **React.js** and **Tailwind CSS**.  
Allows users to browse, create, edit, and manage their blogs seamlessly.

---

### 1️⃣ Install Dependencies

2️⃣ Run the Development Server
